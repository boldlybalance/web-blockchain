# CryptoWeb

Blockchain & Web3 news site for the Boldly Balance network.

**Live:** https://crypto.boldlybalance.life

## Structure

```
├── index.html          # Homepage
├── articles.html       # All articles
├── about.html          # About page
├── contact.html        # Contact page
├── styles.css          # Main styles
├── main.js             # JavaScript
├── favicon.svg         # Favicon
├── robots.txt          # SEO
├── category/
│   ├── bitcoin.html    # Bitcoin category
│   ├── ethereum.html   # Ethereum category
│   └── defi.html       # DeFi category
└── privacy.html        # Legal pages
```

## Deployment

Deployed via Vercel to `crypto.boldlybalance.life` subdomain.
